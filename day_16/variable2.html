Variable

it is used to store data/value and its acts like a container

Based on declaration:

  1. var
    • var is used to declare the variable
    • var is used to redeclare the variable
    • var is used to reinitialize the variable
    • Function scope
  2. let
    • let is used to declare the variable
    • let is used to reinitialize the value
    • block scope
  3. const
    • const is used to declare and assign the variable
    • block scope
